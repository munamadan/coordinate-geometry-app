# Coordinate Geometry Explorer

An interactive web application for exploring coordinate geometry concepts through hands-on visualization and gamified learning.

## Features

### Line Builder
- Drag points on a coordinate grid to explore lines
- Calculate slope, y-intercept, and equation in real-time
- View rise and run visually
- See midpoint between two points

### Slope Racer
- Test your slope knowledge with timed quiz questions
- Multiple difficulty levels (Easy and Medium)
- Score tracking with streak bonuses

### Shape Explorer
- Place points to create triangles and quadrilaterals
- Automatic classification (Equilateral, Isosceles, Scalene, Right, Rectangle, Square, etc.)
- Calculate perimeter, area, side lengths, and slopes

### What Changed?
- Predict how geometric properties change when shapes/lines are modified
- Multiple question types about slopes and areas

## Tech Stack

- HTML5
- CSS3 (with CSS custom properties)
- JavaScript (Vanilla)
- Tailwind CSS (via CDN)
- Google Fonts (JetBrains Mono, Outfit, Space Grotesk)

## Usage

Open `index.html` in a web browser. No build step or server required.

## File Structure

```
index.html    - Main HTML structure
styles.css    - All custom styles
script.js     - Application logic
```

## Browser Support

Works in modern browsers with canvas support.
